Dependencies¶

AnyPyTools currently has the following external dependencies,

Run Time:

  • Python v2.7+/3.4+
  • Numpy
  • Python-Future
  • SciPy*
  • PyDOE*
  • h5py*
  • pywin32 (only Py27)

Note

Packages markerd with (*) are partly optional and only used in specific parts of the library.
