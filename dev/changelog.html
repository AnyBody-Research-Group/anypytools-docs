AnyPyTools Change Log¶

Changed:

  • Improved exception handling when trying to access data which is not avaible in the output.

v0.10.1¶

Changed:

  • Updates and fixes to the documentation website.
  • Added flake8 testing on Travis CI

v0.10.0¶

Merged pull requests:

  • Fix PEP8 issues and remaining pytest issues #21 (melund)
  • Update Documentaion and tutorials #20 (melund)
  • Add SaveData MacroCommand for saving hdf5 files #19 (melund)
  • Fix Crash on Python 2.7 when using h5py_wrapper #18 (melund)
  • Setup Travis-CI for building documentation for publishing on github.io #13 (melund)
  • Refactor the library for the new library documention. #12 (melund)
  • Added AnyPyProcessOutputList.tolist() converting results to native Python #11 (KasperPRasmussen)
